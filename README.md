🧭 Decision Mapping Lite

这是什么
一个将模糊的人生/职业/项目选择，转化为可执行“决策备忘录（Decision Memo）”的自助工具：通过问题重构 → 约束识别 → 可控性分析 → 证据门槛 → 试探路径，帮助你在不确定中恢复可执行的稳定状态。

适用场景
职业转型（如是否转行/读博/换岗）、项目选择（是否推进某个PoC/创业方向）、人生关键决策（offer选择/城市迁移等）。

使用步骤
填写决策表单 → 生成 Decision Memo（含推导链与下一步） → 下载为 Markdown 保存与复盘。

免责声明
本工具提供结构化思考与决策支持，不构成医疗、法律或财务建议。

隐私说明
本工具默认不存储你的输入内容；所有生成结果仅在当前会话中显示，由你自行下载保存。
